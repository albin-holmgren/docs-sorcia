## üìö Content Guidelines

### Writing Style
- **Tone**: Professional but friendly
- **Voice**: Second person ("you")
- **Tense**: Present tense
- **Length**: Scannable sections, short paragraphs

### Code Examples
- Always include multiple languages (cURL, JavaScript, Python)
- Show complete, runnable examples
- Include error handling
- Add comments for clarity

### Visual Elements
- Use callouts (Info, Warning, Tip, Check) liberally
- Break long pages with images/diagrams
- Use tables for comparisons
- Add accordions for optional content

### SEO
- Descriptive titles (50-60 characters)
- Meta descriptions (150-160 characters)
- Proper heading hierarchy (H1 ‚Üí H2 ‚Üí H3)
- Internal linking between related pages
