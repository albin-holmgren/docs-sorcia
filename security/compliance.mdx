**Status**: Certified

Covers:
- Security controls
- Availability guarantees
- Processing integrity
- Confidentiality
- Privacy

**Audit Period**: Annual

## Security Standards

### Encryption

- **At Rest**: AES-256
- **In Transit**: TLS 1.3
- **Backups**: Encrypted
- **Keys**: AWS KMS or customer-managed

### Access Control

- Role-Based Access Control (RBAC)
- Multi-Factor Authentication (MFA)
- Single Sign-On (SSO)
- IP allowlisting

### Monitoring

- 24/7 security monitoring
- Intrusion detection
- Automated alerts
- Incident response plan
